# Computer-Network-LAN-A-Email-Server-Configuration-in-Cisco-Packet-Tracer

A computer LAN featuring two switches and routers connected to eight computers that have three computer users which are able to send and receive email via a server that provides Email services.

Requirements : 

1. Eight computers
2. Two Switches
3. Two Routers
4. Copper straight throw cable
5. Copper Cross throw cable
6. Server

Each system required will be needed to register an email address and password.

On the server side the server can act as an admin which then has the privilege of assigning email domains as well as registering users that will be granted for the email services.
